# 面经 Wiki

本目录是对仓库根目录下"面经原始素材"（公司面经 + `后端技术专栏/` + `Agent技术专栏/` + `项目/`）的二次编译。
**约定：** 仓库根目录下的所有 `.md` 是 raw（原始素材），`wiki/` 是由 LLM 维护的派生层，里面的内容尽量都是从 raw 中提炼、汇总、双链而来。

## 怎么用

- **明天面 X 公司？** 打开 [[公司索引]] → 找到对应公司画像 → 顺着画像里的"考察重点"双链复习。
- **想突击某个八股？** 打开 [[主题索引]]（按技术域分类）或 [[高频考点榜]]（按出现次数排序）。
- **被问到项目？** 打开 [[项目索引]]，每个项目都有一份"被追问过的所有问题 + 标准回答骨架"。
- **算法题？** 打开 [[算法题索引]]，按出现频次排序的真题清单。


## 维护规范（给 Claude 自己看）

1. **wiki 里只写从 raw 里能找到出处的内容**——所有论断尽量挂一个 raw 文件双链。
2. **双链格式**：raw 文件用全路径 `[[字节-今日头条-财经业务/我的字节一面]]`；wiki 内部互链用裸名 `[[MySQL索引与执行计划]]`。
3. **更新触发**：当 raw 目录新增面经 / 技术专栏文档时，应增量更新对应公司画像、主题文章和高频考点榜的频次统计。
4. **拒绝复述**：wiki 里的解答如果在 `后端技术专栏/` 已经有，就只放一句话 + 双链，不重复正文。
5. **保留出处**：每个观点旁标记一下"出现于 X 公司 X 面"，方便面试前复盘是哪家公司挖过这个点。
