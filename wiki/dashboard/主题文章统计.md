# 主题文章统计

> 全部 wiki/技术主题/ 下文章的元数据视图。

## 全部主题（按域分组）

```dataview
TABLE WITHOUT ID
  file.link AS "主题",
  domain AS "技术域",
  length(file.outlinks) AS "外链数",
  length(file.inlinks) AS "被引用次数"
FROM "wiki/技术主题"
WHERE type = "topic"
SORT length(file.inlinks) DESC
```

## 没有 domain 字段的文章（待补 frontmatter）

```dataview
LIST
FROM "wiki/技术主题"
WHERE type = "topic" AND !domain
```

> 这是 `domain` 字段缺失的列表 — 可以自己补一下，让上面的"按域分组"更准。

## 没有被任何文件引用的孤岛主题

```dataview
LIST
FROM "wiki/技术主题"
WHERE type = "topic" AND length(file.inlinks) = 0
```

> 这些文章**没有任何 wiki 文件引用它**——要么是没人在乎的考点（删？），要么是双链没挂好（找出来挂）。

---

## 用法

- 看哪个主题被引用得最多 = 你 wiki 里出现频率最高的考点
- 找孤岛文件——这些是双链坏掉或者真没用的
- 写新主题前，先看 domain 分布，避免重复
