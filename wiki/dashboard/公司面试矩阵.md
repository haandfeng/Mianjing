# 公司面试矩阵

> 公司 × 面经 / 复盘 / 难度的全景视图。

## 全部公司画像（含面经文件数）

```dataview
TABLE WITHOUT ID
  file.link AS "公司画像",
  tier AS "层级",
  direction AS "方向"
FROM "wiki/公司"
WHERE type = "company-profile"
SORT tier ASC, file.name ASC
```

## 复盘数 × 公司

```dataview
TABLE WITHOUT ID
  company AS "公司",
  length(rows) AS "复盘场数",
  rows.verdict AS "各场结果"
FROM "wiki/复盘"
WHERE type = "review"
GROUP BY company
SORT length(rows) DESC
```

## 你已经面过 vs 仅有画像 vs 完全没接触

```dataview
LIST
FROM "wiki/公司"
WHERE type = "company-profile" AND !contains(file.outlinks.path, "复盘")
```

> 这张是"有画像但还没面过的公司"列表，作为投简历清单。

## 用法

- **新增公司投递**：先看 [[索引/公司]]，对比仓库已收集的画像，决定要不要先投这家
- **复盘对比**：第一张表 + 第二张表对照看，能立刻识别"我面过的公司大都什么层级 / 方向"
- **告诉 Claude 帮你**：`帮我列出还没面过的中型公司画像` —— Claude 会读这页给你
