# 最近更新

> wiki 里最近被改过的文件。看 wiki 在长什么 / 你最近在关注啥。

## 最近 7 天

```dataview
TABLE WITHOUT ID
  file.link AS "文件",
  file.mtime AS "修改时间"
FROM "wiki"
WHERE file.mtime > date(today) - dur(7 days)
SORT file.mtime DESC
LIMIT 30
```

## 最近 30 天

```dataview
TABLE WITHOUT ID
  file.link AS "文件",
  file.mtime AS "修改时间",
  type AS "类型"
FROM "wiki"
WHERE file.mtime > date(today) - dur(30 days)
SORT file.mtime DESC
LIMIT 50
```

## 最久没动的文件（90 天+）

```dataview
LIST
FROM "wiki"
WHERE file.mtime < date(today) - dur(90 days)
SORT file.mtime ASC
LIMIT 20
```

> 这是"陈年文件"列表——可能信息过期，定期检查。

---

## 用法

- 看自己最近在关注什么主题（高频出现的 file.link）
- 看长期没动的文件（信息可能过期）
- 给 Claude 看："这周改了什么，帮我做个总结" / "最近哪些公司画像没更新过"

## 时间字段

`file.mtime` 是文件修改时间，由文件系统记录。Obsidian 不会自动改 frontmatter `updated:` 字段——如果你想让 `updated:` 跟 `mtime` 同步，装 [Update time on edit](https://github.com/beaussan/update-time-on-edit-obsidian) 插件。
