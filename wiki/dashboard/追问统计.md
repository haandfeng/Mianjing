# 项目追问统计

> wiki/追问/ 下 5 个项目 Q&A 库的元数据视图。

## 全部项目

```dataview
TABLE WITHOUT ID
  file.link AS "项目",
  project_name AS "项目名",
  period AS "时间",
  length(file.inlinks) AS "被引用次数"
FROM "wiki/追问"
WHERE type = "project"
SORT length(file.inlinks) DESC
```

## 哪些公司深挖了哪个项目（基于 backlinks）

```dataview
TABLE WITHOUT ID
  file.link AS "追问页",
  filter(file.inlinks, (l) => contains(l.path, "公司/")) AS "深挖此项目的公司"
FROM "wiki/追问"
WHERE type = "project"
```

> 这张表在 Dataview 里能看到每个项目页被哪些 `公司/X` 文件引用——等于"哪些公司面试官追问过这个项目"。

---

## 用法

- 投简历前：看哪个项目"覆盖面"广（被多家公司追问的项目=面试时强项）
- 准备某家面试：在表里查公司名，看他们以前问过哪些项目
- 项目讲解优先级：从 inlinks 多的开始练
