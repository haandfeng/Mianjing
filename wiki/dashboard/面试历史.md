# 面试历史

> 所有复盘按日期倒序排。装 Dataview 后自动渲染。

## 全部复盘

```dataview
TABLE
  company AS "公司",
  round AS "轮次",
  date AS "日期",
  verdict AS "结果"
FROM "wiki/复盘"
WHERE type = "review"
SORT date DESC
```

## 按公司分组

```dataview
TABLE WITHOUT ID
  company AS "公司",
  length(rows) AS "轮次数",
  rows.round AS "轮次"
FROM "wiki/复盘"
WHERE type = "review"
GROUP BY company
SORT length(rows) DESC
```

## 本月（2026-05）的复盘

```dataview
LIST
FROM "wiki/复盘"
WHERE type = "review" AND startswith(date, "2026-05")
SORT date DESC
```

## 通过率（含数据时显示）

```dataview
TABLE WITHOUT ID
  verdict AS "结果",
  length(rows) AS "次数"
FROM "wiki/复盘"
WHERE type = "review"
GROUP BY verdict
```

---

## 用法

- 查"我面过哪些公司、哪些轮次"——直接看第一张表
- 查"哪家面得最多" / "哪家挂的多"——看分组表
- 查"这个月面了多少场"——本月那段

## 添加新数据

填 [[复盘/模板]] 或跑 `/new-review 公司 轮次`，新文件会自动被这页索引到。

## 静态备份（无 Dataview 时）

`复盘/` 目录下文件按日期排序的话也基本等价，命令行查：

```bash
ls -t wiki/复盘/*.md | grep -v 模板 | grep -v 示例 | grep -v README
```
