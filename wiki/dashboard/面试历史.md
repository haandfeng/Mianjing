# 面试历史

> 所有面经按时间倒序。**主要看近一年**。

## 近一年（最重要）

```dataview
TABLE WITHOUT ID
  file.link AS "面经",
  company AS "公司",
  bu AS "BU",
  round AS "轮次",
  date AS "日期",
  verdict AS "结果",
  tags AS "标签"
FROM "公司"
WHERE date >= date(today) - dur(365 days) AND type = "面经"
SORT date DESC
```

## 全部历史

```dataview
TABLE WITHOUT ID
  file.link AS "面经",
  company AS "公司",
  date AS "日期",
  verdict AS "结果"
FROM "公司"
WHERE type = "面经"
SORT date DESC
```

## 按公司分组（近一年）

```dataview
TABLE WITHOUT ID
  company AS "公司",
  length(rows) AS "近一年场数",
  rows.verdict AS "各场结果"
FROM "公司"
WHERE type = "面经" AND date >= date(today) - dur(365 days)
GROUP BY company
SORT length(rows) DESC
```

## 通过率统计

```dataview
TABLE WITHOUT ID
  verdict AS "结果",
  length(rows) AS "次数"
FROM "公司"
WHERE type = "面经" AND verdict
GROUP BY verdict
```

## 我面试 vs 网上转载

```dataview
TABLE WITHOUT ID
  source AS "来源",
  length(rows) AS "数量"
FROM "公司"
WHERE type = "面经"
GROUP BY source
```

---

## 用法

- 看"我什么时候面过哪些公司"——第一张表
- 看"哪家面得多 / 挂得多"——按公司分组
- 看"我自己 vs 网上"占比——最后那张

## 没数据怎么办

dashboard 全空是因为：
1. 没装 Dataview → [[插件设置]]
2. 面经文件没 frontmatter → 用 [[模板/录音转面经]] 重新整理或手动加：
   ```
   ---
   type: 面经
   company: 字节
   bu: 财经
   date: 2024-08-15
   round: 一面
   verdict: 挂
   source: 我面试
   tags: [java, mysql]
   ---
   ```
