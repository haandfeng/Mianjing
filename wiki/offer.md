# Offer 决策板

> 面试季每场面完更新一行。Offer 来了集中比较。

## 决策标准（按权重）

1. **成长（最重）**：技术挑战 / 业务深度 / 团队人才密度
2. **技术栈匹配**：是否能延续简历主线（性能 / Agent / 后端）
3. **团队**：直属 leader 风格 / 工作节奏 / 同事质量
4. **薪资**：日薪 / 月薪 / 转正条件
5. **地点**：通勤时间 / 城市生活成本

> 优先级 1-5（5 最强意向）。
