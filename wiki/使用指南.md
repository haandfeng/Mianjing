# 使用指南

> 本仓库 = 面试 + 学习的个人知识库。**4 件事**，做好这 4 件就够。

## 0. 先装插件（5 分钟）

[[插件设置]] —— 必装 Dataview。Templater / Tasks 选装。

## 1. 看 / 找面经

```
Cmd+O 输公司名 → [[公司/字节]] / [[公司/快手]] / ...
↓
看「考察重点」+「真题摘录」
↓
跳双链到 技术主题/ 看八股骨架，或跳到 raw 看完整面经上下文
```

也可以打开 [[dashboard/面试历史]] 看时间线 + 通过状态全表。

## 2. 加新面经（最常用）

**3 种来源 → 1 种处理**：

| 来源 | 入口 |
|---|---|
| 我面试的录音 | [[模板/录音转面经]]（喂 LLM 的提示词模板）|
| 网上 copy 的面经 | [[模板/录音转面经]]（同流程）|
| 自己手敲 | [[模板/新面经原文]] |

完事后：
- 文件放 `公司/<公司>/<BU>/<标题>.md`
- frontmatter 必填：`company` `bu` `date` `verdict`（通过 / 挂 / 未知）+ `tags: [java, mysql, ...]`
- 跑 `/wiki-lint` 验证 0 broken

## 3. 学习 / 复习专题

```
[[索引/主题]] → 找想复习的主题
↓
[[Java并发集合]] 等 → 看答题骨架 + 易错点 + 真题来源
↓
深入：跳到 raw 专栏 [[专栏/后端/Java并发编程]]
```

Agent / RAG 问题 → [[Agent问题答案/README]]（43 题分主题）。

## 4. 加新专题

新主题（出现在 ≥3 家公司就值得建）：
1. raw 写笔记到 `专栏/后端/<主题>.md` 或 `专栏/Agent/<主题>.md`
2. wiki 用 [[模板/新主题文章]] 建 `wiki/技术主题/<主题>.md`（压缩版）
3. 在 [[索引/主题]] 加一行索引
4. 顺手在对应公司画像加双链（互相索引）

## 全局 dashboard

```
[[dashboard/面试历史]]       → 时间 + 公司 + 通过状态
[[dashboard/公司爱考什么]]   → 每家公司高频考点 tag
[[dashboard/主题统计]]       → 主题被引用次数（哪些是高频考点）
```

只有装了 Dataview 才能看到自动渲染。

## 双链规则

- raw 文件全路径：`[[公司/字节/财经/字节财经一面]]`
- wiki 内部裸名：`[[Java并发集合]]`、`[[索引/主题]]`
- 简历 PDF 不进 wikilink（在仓库根 `简历/` symlink）

## Obsidian 快捷键

| 操作 | 快捷键 |
|---|---|
| 按名打开 | `Cmd+O` |
| 全文搜索 | `Cmd+Shift+F` |
| 跳转 wikilink | `Cmd+点击` |
| 反向看引用 | 右侧 Backlinks |
| 看 vault 拓扑 | `Cmd+G` |
| 源 / 预览切换 | `Cmd+E` |
