# 按公司画像反问

> 一公司一段定制反问。每段 3-5 条，结合该公司画像中提到的特点。详情见对应 [[索引/公司索引]] 文件。

## 快手（[[公司画像/快手]]）

- 你们大数据引擎从离线 Spark 迁到实时 Flink，目前最大的技术挑战是性能还是数据正确性？
- Gluten 向量化引擎用 Rust 重写 Java 生态，团队 Rust 经验如何沉淀的？
- 实习从日常转暑期再到秋招的路径常见吗？团队对长期实习的偏好是？
- 大数据团队和算法 / 业务团队的协作节奏？
- 您对实习生最重要的评价维度是工程能力还是业务理解？

## 字节（[[公司画像/字节]]）

- 这条业务线在字节内部的优先级？
- 团队当前最大的技术挑战是规模问题还是产品形态变化快带来的问题？
- 字节 OKR 节奏快，实习生怎么参与目标设定？
- 跨 BU 流动机会多吗？实习期间能切换组吗？
- 你们怎么用 AI Coding 提效（Marscode 内部用得多吗）？

## 腾讯（[[公司画像/腾讯]]）

- 腾讯各事业群（CSIG / WXG / IEG）协作节奏差别大吗？
- 你们这边的中间件 / 基础架构是用集团内部的还是自研？
- 实习生能否参与到核心链路设计 / Code Review？
- 转正流程是组内闭环还是要 BG 统一答辩？
- 业务相对稳定的团队，技术上还有哪些升级空间？

## 蚂蚁（[[公司画像/蚂蚁]]）

- 蚂蚁强金融场景，您团队对稳定性 / 数据一致性的红线是怎么定的？
- 蚂蚁中间件（SOFA 系列）在你们组用得深吗？哪些是必学？
- 实习生在金融场景下能拿到的权限和参与度如何？
- 你们做风控 / 支付 / 计算各方向的技术挑战差别是什么？
- 转正答辩看 input 还是 output？

## 阿里云（[[公司画像/阿里云]]）

- 阿里云这边和集团其他 BU 协作模式是怎样的？
- 云产品迭代节奏和大客户需求节奏怎么平衡？
- 你们组用 Java 还是 Go 多？为什么？
- 实习生有机会参与到客户对接 / 交付吗？
- 阿里云内部对 AI 原生产品的押注力度如何？

## 小红书（[[公司画像/小红书]]）

- 客户端架构方向，AI Coding 提效的具体落地形式是什么？
- 你们对前后端融合的要求高吗？实习生主攻哪一端？
- 卡顿 / Crash / ANR 监控做到什么粒度？
- 团队风格更偏工程师文化还是产品驱动？
- 转正概率和评估时机？

## 微软（[[公司画像/微软]]）

- Bing 图像视频业务的 Agent 化探索具体在做什么场景？
- 实习生更多做线上业务支持还是新功能探索？
- 评估实习生表现更看重工程落地、模型应用还是业务理解？
- 与 Office Copilot / Azure AI 团队的协作多吗？
- WLB 和加班节奏？

## 蔚来（[[公司画像/蔚来]]）

- 车载场景下后端 / 数据团队的工作和互联网公司差别在哪？
- 实时数据链路和车端如何配合？
- 蔚来作为新势力，技术债集中在哪？
- 实习生是否有机会接触车端 / 云端联调？
- 团队对全栈能力的要求？

## TT / TikTok（[[公司画像/TT-TikTok]]）

- 海外业务和国内业务的技术栈差异？
- SRE / 平台方向的工作多大比例是 on-call？
- 跨时区协作节奏怎么样？
- 你们的稳定性指标（SLO）是怎么定的？
- 实习生有机会参与海外项目吗？
