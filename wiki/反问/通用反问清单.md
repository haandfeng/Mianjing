# 通用反问清单

> 任意场可问的备选反问，按场景分类。每场挑 2-3 条即可，避免列清单。

## 破冰类（开局或岗位介绍后）

1. 您在团队多久了，目前主要带哪条线？
2. 团队最近半年最重要的一件事是什么？
3. 团队规模和分工（前后端/算法/数据各占多少）？
4. 这个岗位是新增 HC 还是替补？
5. 团队和 X 团队（上游/下游）怎么协作？

## 业务理解类（项目/业务深挖后）

6. 你们当前业务最大的技术挑战是什么？
7. 接下来半年的技术规划重点是什么方向？
8. 这块业务的衡量指标（北极星指标）是什么？
9. 当前最痛的瓶颈是规模、性能还是稳定性？
10. 业务在公司内的优先级是怎样的？

## 技术深挖类（面试官提技术后）

11. 刚才你提到 X 技术，能说说当时为什么选它而不是 Y 吗？
12. 这个系统架构的核心难点是什么？
13. 技术债主要集中在哪一块？
14. 有计划做技术升级 / 重构吗？
15. 团队的技术分享 / Code Review 机制怎么样？
16. 你们怎么用 AI / Copilot 提效？团队里大家用得多吗？
17. 上下游接口稳定性怎么保证？

## 团队文化类

18. 团队加班节奏怎么样？大版本 vs 平时？
19. 实习生 / 新人 onboarding 一般多久能独立做事？
20. 团队对失败容忍度如何？比如线上事故的处理文化？
21. 跨部门 / 跨地域协作多吗？
22. 团队成长的人都去了哪里 / 做什么？

## 转正 / 留任类

23. 实习生评估什么时候开始？看哪些维度？
24. 转正比例大致是多少？
25. 实习期间通常做哪些项目能算"亮眼"？
26. 您觉得我加入后第一个月会做什么？
27. 转暑期实习能延伸到秋招吗？

## 面试反馈类（结尾必问）

28. 您对我今天的表现有什么建议？
29. 哪些方面您觉得我可以重点提升？
30. 您觉得我和这个岗位的匹配度怎么样？

## 用法

- **HR 面 / 一面**：1-2-5-23-28
- **技术二面**：6-11-13-15-28
- **Leader 面 / 终面**：7-9-22-26-29
