# 可视化入口

> 用 Mermaid 把抽象的"自评 / 时间线 / 主题地图 / 公司流程"做成图，便于复习和外推。

## 文件

- [[技能雷达]] — 十维度自评（mindmap / pie）
- [[简历时间线]] — 实习 + 教育 gantt
- [[技术主题地图]] — 22 个主题分组 mindmap
- [[公司面试流程图]] — 代表公司面试轮次 flowchart

## 渲染

Obsidian 默认支持 mermaid 代码块渲染（mindmap、gantt、flowchart、pie）。
不支持 radar 图，所以技能自评用 mindmap + pie 替代。

## 用法

- 面试前一晚扫一眼 [[简历时间线]] 复习时间线话术
- 面试季中对照 [[公司面试流程图]] 安排时间
- 复习准备时按 [[技术主题地图]] 找薄弱区
