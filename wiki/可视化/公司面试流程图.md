# 公司面试流程图

> 代表公司的面试轮次和典型周期。具体每家见 [[索引/公司索引]]。

## 字节

```mermaid
flowchart LR
    A[投递] --> B[一面 60min]
    B --> C[二面 60min]
    C --> D[三面 / Leader 60min]
    D --> E[HR 30min]
    E --> F[Offer]
    B -. 1-3 天 .-> C
    C -. 1-3 天 .-> D
    D -. 几天 .-> E
```
- 节奏快，全程通常 1-2 周
- 实习 / 暑期：3 轮技术 + HR；秋招：可能加交叉面

## 快手

```mermaid
flowchart LR
    A[投递] --> B[一面 60min]
    B --> C[二面 60min]
    C --> D[Leader / HR 30-60min]
    D --> E[Offer]
    B -. 1-2 周 .-> C
```
- 整体偏稳，2-3 轮
- 大数据 / 引擎方向技术深度高

## 腾讯

```mermaid
flowchart LR
    A[投递] --> B[一面 60min]
    B --> C[二面 60min]
    C --> D[GM / 总监面 30-60min]
    D --> E[HR 面 30min]
    E --> F[Offer]
    B -. 1-2 周 .-> C
    C -. 1-3 周 .-> D
```
- 周期较长，2-4 周
- 总监面权重大，BG 间差异明显（CSIG / WXG / IEG）

## 蔚来

```mermaid
flowchart LR
    A[投递] --> B[一面 60min]
    B --> C[二面 / 跨部门 60min]
    C --> D[HR 30min]
    D --> E[Offer]
```
- 流程较轻，2-3 轮
- 车端与互联网思路结合，跨部门协作多
