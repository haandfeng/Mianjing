# 工作流（按时间触发）

wiki 是字典，工作流是流水线。每份清单都直接 actionable，照着做就行。

| 触发时机 | 用哪份 |
|---|---|
| 收到面试通知，距面试 ≥ 7 天 | [[考前1周]] |
| 距面试 ≤ 24 小时 | [[考前1天]] |
| 距面试 ≤ 1 小时 | [[考前1小时]] |
| 面试刚结束 30 分钟内 | [[复盘/模板]]（一行命令开新文件：Claude Code 里 `/new-review`，或 `bash wiki/scripts/new_review.sh 字节 一面`）|
| 每周日晚上 | [[周末复盘]] |
| 加新面经入仓 | [[添加新面经]] —— SOP + 双链规范 |

## 总图

```
通知到 ──> 考前1周（系统复习）──┐
                               ├──> 面试 ──> 复盘 ──┐
        考前1天（针对性强化）──┤                   │
                               │                   ▼
        考前1小时（最后预热）──┘             周末复盘（更新自评 + offer 板）
                                                   │
                                                   ▼
                                              下周一新一轮
```
