# 考前 1 天

> 不学新东西。只针对性强化 + 心态调整。

## 上午（2 小时）— 公司画像 + 反问

- [ ] [[索引/公司]] → 打开该公司画像
- [ ] **真题摘录**全部背一遍（如果是数据 / 数字 类，重点记数字）
- [ ] [[反问/按公司]] 找该公司的定制反问，准备 3 条
- [ ] 加 1 条 "结尾问反馈" 类反问（万能）

## 下午（2 小时）— 项目 + 简历对齐

- [ ] [[索引/追问]] 找该公司可能问的项目（看简历相关性 + 公司画像里被问过哪些项目）
- [ ] 每个项目过一遍 STAR 自陈，**对照 [[我的简历/README]] 数字记牢**：
  - 西门子：76% → 92%, 嵌入 ×2, 4096 维, F1 ≈ 0.8
  - 用友：8K+ 员工, 缓存命中率 90%, QPS -1K, p95 -40%
  - 黑马点评：400ms → 100ms
  - 矩阵乘法：ARM 22 GFLOPS（朴素 ×20）/ CUDA 3.6-3.8 TFLOPS（cuBLAS 80%）
- [ ] 看 [[我的简历/差异表]]，确保口径统一

## 晚上（1 小时）— 算法预热

- [ ] 看 [[算法/README]] 中该公司出过的题，每道动手写一遍
- [ ] 看一遍 [[算法/README]] "复习重点"
- [ ] 打开 [[算法/README]] 中可能用到的 1-2 个模板文件，能默写

## 睡前 30 分钟

- [ ] **不再学习**，过一遍 [[我的简历/基础信息]] 自陈一句话
- [ ] 准备好明天的设备 / 简历 PDF / 反问清单
