# Obsidian 插件设置

> 本仓库设计时**默认你装了 3 个核心插件**。没装也能用（只是所有 dashboard 看不到数据），强烈推荐装一遍。

## 必装（3 个）

### 1. Dataview ★

**作用**：把 wiki 当数据库查。本仓库的所有 dashboard 都用它。

**装法**：
1. Obsidian → Settings → Community plugins → Browse
2. 搜 `Dataview` → Install → Enable
3. （可选）Settings → Dataview → 打开 "Enable JavaScript Queries"（部分高级 dashboard 用得到）

**装完做这一步立刻见效**：打开 [[dashboard/README]]，所有表格都会自动渲染。

**学习成本**：5 分钟看懂 [[dashboard/README]] 里的几个例子就够日常用，不需要看官方文档。

### 2. Templater ★

**作用**：插入模板时自动填日期、文件名、变量。本仓库 [[模板/README]] 里 5 份模板都设计成 Templater 兼容。

**装法**：
1. Community plugins → 搜 `Templater` → Install → Enable
2. Settings → Templater → "Template folder location" 填 `wiki/模板`
3. （可选）配快捷键：Settings → Hotkeys → 搜 `Templater: Insert template` → 绑定 `Cmd+Shift+T`

**装完用法**：新建空文件 → `Cmd+Shift+T` → 选模板 → 自动填好。

> 不装 Templater 也能用模板——手动复制 [[模板/README]] 下的文件改名即可。Templater 只是省 30 秒的事。

### 3. Tasks（可选但推荐）

**作用**：把 wiki 里所有 `- [ ]` 复选框聚合成一张待办看板。[[工作流/README]] 里大量 checkbox 全靠它统一管理。

**装法**：
1. Community plugins → 搜 `Tasks` → Install → Enable
2. 设置默认状态：Settings → Tasks → "Set Done Date" → On

**用法**：见 [[dashboard/TODO汇总]]——会自动列出所有未完成任务，按到期日排序。

## 选装（按需）

### Calendar

**作用**：左侧栏给你一个日历。点某天 → 创建当天的 Daily Note 或跳到那天的 [[复盘/模板]] 文件。

**装法**：搜 `Calendar` → Install。配合默认的 Daily Notes 插件用。

**对本仓库用处**：复盘文件命名是 `YYYY-MM-DD-公司-轮次.md`，calendar 能可视化你哪天面了哪场。

### Excalidraw

**作用**：手绘图表，比 Mermaid 灵活。

**仓库已有内容**：`Excalidraw/` 目录已有几张系统设计图。

**装法**：搜 `Excalidraw` → Install → 重启 Obsidian。

### Recent Files

**作用**：左侧栏显示最近打开的文件。

**仓库已有**：`.obsidian/plugins/recent-files-obsidian/` 配置已存在，启动 Obsidian 时会被加载。

### Mermaid（内置）

**不用装**——Obsidian 内置渲染。本仓库 [[可视化/README]] 下的 7 张图都用 Mermaid。

## 装完后第一件事

依次打开下列 5 个文件，确认能看到完整渲染：

1. [[使用指南]] —— 所有快捷键 + 工作流
2. [[dashboard/README]] —— Dataview 表格自动列出所有公司/复盘/项目/Offer 状态
3. [[模板/README]] —— Templater 触发时显示的模板列表
4. [[可视化/wiki鸟瞰图]] —— Mermaid 渲染整体结构
5. [[索引/高频考点榜]] —— 自评列你可以直接编辑

如果有任何一项没正常渲染，去插件设置里开 / 关一下重启。
