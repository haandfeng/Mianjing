# 算法模板入口

> 设计目标：面试当下能照着默写。每份模板 = 最简骨架 + 关键变式题 + 双链回真题出处。

## 模板列表

- [[滑动窗口模板]] — 双指针通用框架；最小覆盖子串 / 限流滑窗 / 子串字符计数
- [[链表模板]] — 反转 / 有环 / 入环 / 倒数N / 双向链表
- [[二叉树模板]] — 前中后序（递归+迭代）/ 层序 / BST验证 / LCA / 对称
- [[图与拓扑模板]] — 邻接表 / DFS / BFS / Kahn / 三色 / 岛屿
- [[DP模板]] — 1D/2D / 零钱 / 01背包 / 丑数 / 股票
- [[设计题模板]] — LRU / LFU / 阻塞队列 / 多线程交替
- [[字符串模板]] — KMP / 大数相加 / atoi / 最长公共前缀

## 关联

- 真题出处归集：[[算法题索引]]
- 高频出现的题：[[高频考点榜]]

## 用法

1. 面试前一晚按公司画像挑 2-3 个模板复默
2. 面试中遇到题目先回忆模板骨架，再套变式
3. 面完把出现的新题挂回[[算法题索引]]并 review 模板
