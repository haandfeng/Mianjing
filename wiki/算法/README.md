## 复习重点（面试当下最有效）

1. **图 + 拓扑** —— 数据平台 / 流处理岗高频。Kahn + DFS 三色法两套都写熟 → [[图与拓扑模板]]
2. **滑动窗口** —— 字符串题入口。LC 76 / 3 / 159 三道把模板抄死 → [[滑动窗口模板]]
3. **链表常考四件套** —— 反转、有环、入环点、合并 K 个 → [[链表模板]]
4. **设计题** —— LRU、LFU、Twitter 时间线、抢红包；不现场推，背最简版本 → [[设计题模板]]
5. **概率题** —— Rand(A)→Rand(B) 一题打通水库抽样、二进制构造 → [[后端技术专栏/用 Rand(A) 实现 Rand(B)]]

> 出现率 >50% 的真题模式：**1 道中等图 / 字符串 + 1 道简单链表 / 数组**。提前 5 分钟预热模板比临场推导快。

## 用法

1. **考前一晚** 按公司画像挑 2-3 个模板复默
2. **考前一小时** [[工作流/考前1小时]] 中 5 道经典模板预热
3. **面试中** 遇题先回忆模板骨架，再套变式
4. **面试后** 把出现的新题加进本文 + review 模板
