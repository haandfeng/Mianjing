# 二叉树模板

## 1. 三序遍历（递归）

```java
void dfs(TreeNode root) {
    if (root == null) return;
    // 前序：访问 root
    dfs(root.left);
    // 中序：访问 root
    dfs(root.right);
    // 后序：访问 root
}
```

## 2. 中序迭代（栈）

```java
Deque<TreeNode> stk = new ArrayDeque<>();
TreeNode cur = root;
while (cur != null || !stk.isEmpty()) {
    while (cur != null) { stk.push(cur); cur = cur.left; }
    cur = stk.pop();
    visit(cur);
    cur = cur.right;
}
```

## 3. 层序（BFS）

出处：[[小鹅通往年面经+我的面经/小鹅通二面面经（结果未知）Cider一面面经（通过）]]

```java
Queue<TreeNode> q = new LinkedList<>();
q.offer(root);
while (!q.isEmpty()) {
    int size = q.size();
    for (int i = 0; i < size; i++) {
        TreeNode n = q.poll();
        if (n.left != null) q.offer(n.left);
        if (n.right != null) q.offer(n.right);
    }
}
```

## 4. 验证 BST（LC 98）

```java
boolean valid(TreeNode root, Long min, Long max) {
    if (root == null) return true;
    if (root.val <= min || root.val >= max) return false;
    return valid(root.left, min, (long)root.val)
        && valid(root.right, (long)root.val, max);
}
```

## 5. LCA（LC 236）

出处：[[快手/支付方向/我的快手支付一面]]

```java
TreeNode lca(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;
    TreeNode l = lca(root.left, p, q);
    TreeNode r = lca(root.right, p, q);
    if (l != null && r != null) return root;
    return l != null ? l : r;
}
```

## 6. 对称二叉树

出处：[[字节-今日头条-财经业务/我的字节一面]]

```java
boolean sym(TreeNode a, TreeNode b) {
    if (a == null && b == null) return true;
    if (a == null || b == null) return false;
    return a.val == b.val && sym(a.left, b.right) && sym(a.right, b.left);
}
```

## 易错点

- BST 验证用区间法，不要只比较父节点
- LCA 递归返回值的含义：在子树中找到 p 或 q 就返回
- 中序迭代左走到底再 pop
