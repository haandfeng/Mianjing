# 图与拓扑模板

## 1. 邻接表存图

```java
List<List<Integer>> g = new ArrayList<>();
for (int i = 0; i < n; i++) g.add(new ArrayList<>());
for (int[] e : edges) g.get(e[0]).add(e[1]); // 有向
```

## 2. DFS 遍历

```java
boolean[] vis = new boolean[n];
void dfs(int u) {
    if (vis[u]) return;
    vis[u] = true;
    for (int v : g.get(u)) dfs(v);
}
```

## 3. BFS 遍历

```java
Queue<Integer> q = new LinkedList<>();
q.offer(start); vis[start] = true;
while (!q.isEmpty()) {
    int u = q.poll();
    for (int v : g.get(u)) if (!vis[v]) { vis[v] = true; q.offer(v); }
}
```

## 4. Kahn 拓扑排序（入度法）

出处：[[快手/大数据平台方向/快手一面面经]]（DAG 合法遍历顺序）

```java
int[] indeg = new int[n];
for (int u = 0; u < n; u++) for (int v : g.get(u)) indeg[v]++;
Queue<Integer> q = new LinkedList<>();
for (int i = 0; i < n; i++) if (indeg[i] == 0) q.offer(i);
List<Integer> order = new ArrayList<>();
while (!q.isEmpty()) {
    int u = q.poll();
    order.add(u);
    for (int v : g.get(u)) if (--indeg[v] == 0) q.offer(v);
}
// order.size() < n -> 存在环
```

## 5. 三色法 DFS 拓扑（检测环）

```java
// 0=未访问 1=访问中 2=已完成
int[] color = new int[n];
boolean dfs(int u) {
    if (color[u] == 1) return false; // 环
    if (color[u] == 2) return true;
    color[u] = 1;
    for (int v : g.get(u)) if (!dfs(v)) return false;
    color[u] = 2;
    order.add(0, u); // 头插，逆后序
    return true;
}
```

## 6. 岛屿数量（LC 200）

出处：[[微软/Bing图像视频业务实习一面面经]]

```java
int count = 0;
for (int i = 0; i < m; i++) for (int j = 0; j < n; j++) {
    if (grid[i][j] == '1') { count++; sink(i, j); }
}
void sink(int i, int j) {
    if (i < 0 || j < 0 || i >= m || j >= n || grid[i][j] != '1') return;
    grid[i][j] = '0';
    sink(i+1,j); sink(i-1,j); sink(i,j+1); sink(i,j-1);
}
```

## 易错点

- Kahn 拿不到全部节点说明有环
- 三色法 1 表示在递归栈中
- 岛屿沉岛时记得改原图避免重复
