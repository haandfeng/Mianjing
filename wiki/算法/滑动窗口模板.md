# 滑动窗口模板

## 通用骨架（双指针）

```java
int left = 0, right = 0;
Map<Character, Integer> window = new HashMap<>();
int valid = 0; // 满足条件的字符数
while (right < s.length()) {
    char c = s.charAt(right++);
    // 1. 扩窗：更新 window 状态
    // 2. 判断是否要收缩
    while (needShrink(window)) {
        char d = s.charAt(left++);
        // 3. 缩窗：更新 window 状态
    }
    // 4. 在合适位置更新答案
}
```

要点
- `right` 进窗口先更新统计；`left` 出窗口后更新统计
- 答案更新时机：扩窗后 / 缩窗前，取决于"求最大"还是"求最小"
- 求最小覆盖：满足条件后开始缩；求最长无重复：违反条件时收缩

## 变式题

### 1. 最小覆盖子串（LC 76）
出处：[[TT/TT SRE]]
- 用 `need` 记录 t 中每个字符的需要数
- `valid == need.size()` 时尝试缩窗
- 缩窗时记录最小窗口起点和长度

### 2. 子串中连续 'A' 出现次数 > k
出处：[[快手/支付方向/快手日常实习（支付方向）面经]]
- 维护当前最长连续 A 的计数 `curRun`
- 遇到非 A 时 `curRun = 0`；遇到 A `curRun++`
- 一旦 `curRun > k` 触发条件，统计有效子串数

### 3. 限流滑动窗口（计数器实现）
- 队列存最近一秒/分钟的请求时间戳
- 新请求来时，队头过期的弹出，比较队列长度和阈值

## 易错点

- `right++` 和 `left++` 写在哪一步：建议先取字符再 `++`
- 窗口大小 = `right - left`（左闭右开）
- 缩窗判断要用 while 不是 if
