# 设计题模板

## 1. LRU（双向链表 + HashMap）

出处：[[字节-今日头条-财经业务/字节-国际广告and中国广告/字节跳动中国商业化一面面经]]、[[小鹅通往年面经+我的面经/小鹅通一面面经（带答案已OC）]]

```java
class LRUCache {
    class Node { int k, v; Node prev, next; Node(int k, int v){this.k=k;this.v=v;} }
    Map<Integer, Node> map = new HashMap<>();
    Node head = new Node(0,0), tail = new Node(0,0);
    int cap;
    LRUCache(int c){ cap=c; head.next=tail; tail.prev=head; }
    int get(int k){ if(!map.containsKey(k))return -1; Node n=map.get(k); remove(n); addHead(n); return n.v; }
    void put(int k, int v){
        if(map.containsKey(k)){ Node n=map.get(k); n.v=v; remove(n); addHead(n); return; }
        if(map.size()==cap){ map.remove(tail.prev.k); remove(tail.prev); }
        Node n = new Node(k,v); map.put(k,n); addHead(n);
    }
    void addHead(Node n){ n.next=head.next; n.prev=head; head.next.prev=n; head.next=n; }
    void remove(Node n){ n.prev.next=n.next; n.next.prev=n.prev; }
}
```

## 3. 阻塞队列

```java
class BlockingQueue<T> {
    Queue<T> q = new LinkedList<>();
    int cap;
    Lock lock = new ReentrantLock();
    Condition notFull = lock.newCondition(), notEmpty = lock.newCondition();
    void put(T t) throws InterruptedException {
        lock.lock();
        try { while (q.size() == cap) notFull.await(); q.offer(t); notEmpty.signal(); }
        finally { lock.unlock(); }
    }
    T take() throws InterruptedException {
        lock.lock();
        try { while (q.isEmpty()) notEmpty.await(); T t = q.poll(); notFull.signal(); return t; }
        finally { lock.unlock(); }
    }
}
```
要点：用 `while` 防虚假唤醒；signal 对应的条件。

## 4. 多线程交替打印

出处：[[腾讯/数据平台/我的腾讯一面]]

```java
// ABC 三线程交替打印，用 ReentrantLock + 3 个 Condition
Lock lock = new ReentrantLock();
Condition[] c = { lock.newCondition(), lock.newCondition(), lock.newCondition() };
int state = 0; // 0=A 1=B 2=C
void print(int id, String s) {
    for (int i = 0; i < N; i++) {
        lock.lock();
        try { while (state != id) c[id].await(); System.out.println(s); state = (id+1)%3; c[(id+1)%3].signal(); }
        finally { lock.unlock(); }
    }
}
```
也可用 Semaphore 轮转。

## 易错点

- LRU 注意 put 已存在 key 时要更新值并移动
- LFU minFreq 维护是难点
- 阻塞队列 signal 唤醒对应 Condition，不要错配
- 交替打印 while 循环判断 state，避免虚假唤醒
