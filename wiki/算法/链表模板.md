# 链表模板

## 1. 反转链表

```java
ListNode prev = null, cur = head;
while (cur != null) {
    ListNode next = cur.next;
    cur.next = prev;
    prev = cur;
    cur = next;
}
return prev;
```

## 2. 是否有环（快慢指针）

出处：[[小红书/小红书产品工程师一面面经]]

```java
ListNode slow = head, fast = head;
while (fast != null && fast.next != null) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow == fast) return true;
}
return false;
```

## 3. 入环点（Floyd）

出处：[[字节/字节财经一面]]

```java
// 相遇后，slow 回到 head，两指针每次走一步，再次相遇即入环点
while (head != slow) { head = head.next; slow = slow.next; }
return head;
```

数学：相遇时 fast 比 slow 多走一圈；head 到入环点 = 相遇点继续走到入环点。

## 4. 删除倒数第 N 个

```java
ListNode dummy = new ListNode(0, head);
ListNode fast = dummy, slow = dummy;
for (int i = 0; i < n; i++) fast = fast.next;
while (fast.next != null) { fast = fast.next; slow = slow.next; }
slow.next = slow.next.next;
return dummy.next;
```

## 5. 双向链表 add/remove（LRU 用）

出处：[[快手/支付方向/快手日常实习面经]]

```java
class Node { int k, v; Node prev, next; }
// 头哨兵 head <-> ... <-> tail 尾哨兵
void addToHead(Node n) {
    n.prev = head; n.next = head.next;
    head.next.prev = n; head.next = n;
}
void remove(Node n) {
    n.prev.next = n.next;
    n.next.prev = n.prev;
}
```

## 易错点

- 头哨兵 `dummy` 简化删头节点逻辑
- 反转链表先存 next 再改指针
- 双向链表 add/remove 顺序：先连新指针，再断旧指针
